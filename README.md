### To-Do list ##

Frontend: REACT com Material UI (Ver Material do Google tbm)
Backend: Java com Spring Boot e Spring data usando banco de dados H2

## Banco de Dados ##
```
CREATE TABLE Todo_List (
	id int,
	description varchar(200),
	done int(1),
	PRIMARY KEY (id)
);
```
